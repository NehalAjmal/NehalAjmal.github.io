# Nehal Ajmal - Personal Portfolio

Welcome to my personal portfolio repository! This is a unique, lightweight, and visually dynamic web portfolio designed to showcase my skills as a Computer Science Engineering student and AI/ML Engineer.

## Live Preview
You can check out the live demo of the portfolio here: **[nehalajmal.github.io](https://nehalajmal.github.io)**

---

## 🎨 Design & Architecture
This portfolio was built from scratch with a focus on simplicity, speed, and modern aesthetics. It deliberately avoids heavy javascript frameworks or complex build tools.

### Features:
- **No Build Step Required**: Built with 100% Pure HTML and Vanilla CSS. It works directly in the browser.
- **Deep Glassmorphism Aesthetic**: Modern glass-like UI cards with blurred backdrop filters and sleek neon gradients.
- **Floating Background Animations**: A dynamic background featuring floating tech logos (Python, TensorFlow, PostgreSQL, etc.) and blurred orbs.
- **Responsive Layout**: Designed to look great on both desktop and mobile devices.

### Technologies Used:
- **HTML5**: Semantic structure.
- **CSS3 (Vanilla)**: All styling, layout grids, CSS variables, and keyframe animations.
- **Vanilla JavaScript**: Minimal scripts for scroll reveals, typing effects, and smooth scrolling.
- **Devicon & FontAwesome**: Vector graphics for technology logos and UI icons.

## 🚀 Getting Started

Because there is no build process, setting up this project locally is incredibly simple.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/NehalAjmal/NehalAjmal.github.io.git
   ```
2. Navigate to the directory:
   ```bash
   cd NehalAjmal.github.io
   ```
3. **Run the site**:
   Simply open the `index.html` file in your preferred web browser. No `npm install` or local servers are required!
